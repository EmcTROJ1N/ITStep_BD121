1. Пользователь вводит имя файла, программа подсчитывает общую сумму чисел в файле
2. Пользователь вводит 2 имени файла, программа копирует из первого файла в второй все числа (отдельные слова) без повторений
3. Пользователь вводит 2 имени файла, программа размещает в каждом предложении (.) слова в обратном порядке и сохраняет текст во второй файл.
4. Пользователь вводит 2 имени файла, программа сортирует все слова по длине и сохраняет их во второй файл
5. Пользователь вводит имя файла (программа на Java), программа выравнивает код "лесенкой"
