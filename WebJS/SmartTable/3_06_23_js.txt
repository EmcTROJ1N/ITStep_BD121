 - item (надпись пункта меню)

 - code (ссылка на функцию-обработчик, запускающуюся при выборе пункта меню)
 
 Разработать класс Menu, который позволяет создать пользовательское меню и имеет следующие члены класса:
 - items[] (коллекция пунктов меню)
 - x, y (координаты меню)
 - show (показать меню)
 - hide (скрыть меню)
 - addItem(item) - добавить пункт меню
 - removeItem(pos) - удалить пункт меню по позиции
 
 Пример использования:
 
	var menu = new SmartMenu(50, 50, 50, 50)
	menu.addItem(new MenuItem('File', '../Images/admin.gif', function () { console.log('file action!') }))
	menu.addItem(new MenuItem('Edit', '../Images/enotik.gif', function () { console.log('edit action!') }))
	menu.addItem(new MenuItem('View', '../Images/enotik.gif', function () { console.log('view action!') }))
	//menu.print()

	menu.show()
