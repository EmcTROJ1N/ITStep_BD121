NodeJS
1. Пользователь вводит путь к папке, программа удаляет из неё все картинки (jpg, png, gif, bmp)

2. Пользователь вводит 2 пути к папкам, программа копирует все текстовые файлы из первой папки во вторую

3. Пользователь вводит имя папки, программа следит за папкой и подпапками и, если там появляются новые текстовые файлы, они
удаляются, а новые картинки копируются в папку BackUp. Все произведенные действия записываются в файл result.log

Формат файла:

12.06.2023 23:23 deleted: test.txt
12.06.2023 23:34 copied: test.gif

-----------------------------------------

4. На первой странице пользователь указывает размеры таблицы умножения,
на второй странице сайт показывает саму таблицу умножения (красиво оформленную стилями)

5. Разработать сайт на Express, который на главной странице показывает список файлов в определённой папке в виде ссылок.
Пользователь может щёлкнуть по любой ссылке на файл. В этом случае файл скачивается на клиентский компьютер.

6. Разработать фотогалерею, которая на главной странице показывает миниатюры картинок (div, flex, table???).
При щелчке на картинку пользователь попадает на страницу картинки, где она показывается в полном размере
и показывается инфо картинки. Пользователь может загружать новые картинки на сервер.
Доп. задание: сервер позвоялет разделять картинки по категориям. Каждая категория хранится в отдельной папке.
