1. Пользователь вводит размеры 2D массива и сам массив, программа копирует во второй одномерный массив только
 отрицательные чётные элементы первого

 2. Пользователь вводит размеры и сами 2 одномерных динамических массива. Программа копирует в третий массив
 только те элементы, которые есть и в первом и во втором массивах БЕЗ ПОВТОРЕНИЙ.
 
 Пример:
 1 2 3 4 5
 1 2 3 3
 Ответ: 1 2 3
 
 3. Пользователь вводит символы, пока не нажмёт Enter. Программа подсчитывает количество не буквенно-цифровых
 символов
 
 4. Пользователь вводит символы, пока не будет введена хотя бы одна цифра. Программа подсчитвает количество
 заглавных букв
