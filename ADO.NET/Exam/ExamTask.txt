Требования к работе
Нужно использовать 2 различные технологии для решения разных задач (хотя бы в одной задаче нужно использовать Entity Framework)
В каждой задаче нужно использовать хранимые процедуры
Таблицы должны быть заполнены данными
Для каждой таблицы реализовать добавление, редактирование и удаление CRUD (в отдельном окне)
Хотя бы в одной задаче использовать WPF

Дополнительные требования ко 2 задаче:
Реализовать сортировку и фильтрацию и группировку элементов хотя бы для одной таблицы
Реализовать поиск по содержимому столбца в разных таблицах
Реализовать сводную таблицу, показывающую данные из нескольких таблиц (как минимум одну)
Проверка целостности БД и правильности вводимых данных
Обязательное использование хранимых процедур
Реализовать максимально удобный для пользователя интерфейс
Хотя бы в одном месте использовать редактирование внутри таблицы.
Хотя бы в одной из таблиц нужно отображать изображение какой-либо сущности.
В БД может храниться адрес картинки, а сами изображения помещаются в отдельную папку. Пользователь может назначать и менять картинку
